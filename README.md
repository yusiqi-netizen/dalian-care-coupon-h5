# 大连市养老服务消费补贴 H5

静态 H5 预览项目，包含：

- `user.html`：用户端指引视频页
- `org.html`：机构端指引视频页

## 部署

项目使用 GitHub + Vercel 工作流。后续修改提交到同一个 GitHub Repository 后，由同一个 Vercel 项目自动重新部署。

## 视频说明

原始视频 `assets/user-guide.mp4` 体积超过 GitHub 普通仓库单文件限制，暂不提交到仓库。正式部署时建议使用压缩后的 Web 版本视频，或将视频放到 Vercel Blob / OSS / CDN 后在页面中引用 HTTPS 地址。
